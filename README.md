# Low-Latency Keccak

This repository contains the Verilog code of low-latency masked Keccak-f[200] implementation 
(SCA-secure hardware implementation) for the paper 

"Low-Latency Keccak at any Arbitrary Order" (TCHES 2021, Issue 4).
